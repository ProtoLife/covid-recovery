Main Matlab script: 'Fit_GGM_zika_supp.m'

This Matlab script contains code for:

1) Fitting the GGM model to early epidemic phase
2) Derive parameter uncertainty
3) Generate short-term forecasts

Uses case incidence data from file:'zika-daily-onset-antioquia.txt' describing the Zika outbreak in Antioquia, Colombia (see: https://www.ncbi.nlm.nih.gov/pubmed/27366586)
